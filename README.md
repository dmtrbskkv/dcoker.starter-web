# Docker starter
A comfortable starter for working in the web

## Included

1. PHP 8.0
2. Nginx latest
3. Node 16
4. PHPMyAdmin latest
5. Mysql 5.7

## How to
**First you need to build images**
```bash
./bin/build.sh
```

**Then you have 2 ways to get started**

Develop mode
```bash
./bin/start-dev.sh
```

Production mode
```bash
./bin/start-dev.sh
```

**That all!**
